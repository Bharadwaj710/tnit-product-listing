# TNIT2026 - Responsive Product Listing & Form Handling

This is a responsive single-page application built with Next.js, fulfilling the TNIT FSD Intern assignment requirements.

## Features

- **Product Listing**: Fetches and displays products from the FakeStoreAPI (`https://fakestoreapi.com/products`).
- **Responsive Grid**:
  - Desktop: 4 columns
  - Tablet: 2 columns
  - Mobile: 1 column
- **Newsletter Form**:
  - Validates Name and Email fields.
  - Logs the submitted data to the browser console.
- **Premium UI**: Uses TailwindCSS for a sleek, modern, and highly responsive design.

## Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Development Stack

- Framework: Next.js (App Router)
- Styling: Tailwind CSS
- Data Fetching: React Server Components & Native Fetch API
